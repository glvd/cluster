## Install

You can import `github.com/glvd/cluster/api/rest/client` in your code.

The code can be downloaded and tested with:

```
$ go get -u -d github.com/glvd/cluster
$ cd $GOPATH/src/github.com/glvd/cluster/rest/api/client
$ go test -v
```
